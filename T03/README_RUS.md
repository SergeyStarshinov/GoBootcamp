# Backend часть веб-приложения на языке **Go** с использованием net/http. 

1. Приложение реализует игру "Крестики-Нолики" человека (Х) с компьютером (О).
2. Реализовано два варианта начала игры, когда первый ход делает человек или компьютер.
3. Для вычисления хода за компьютер используется алгоритм «Минимакс»;
4. Программа написана с использованием библиотек net/http и uber-fx.
5. Хранение текущих игр производится в оперативной памяти, в sync.Map
6. Код программы разделен на слои: 
    - datasource (работа с данными)
    - domain (бизнес-логика)
    - web (взаимодействие с клиентом)
    - di (конфигурация для внедрения зависимостей)
7. Обрабатываемые запросы от клиента:
    - `GET /new_game` - создание новой игры с пустым полем, первый ход делает игрок
    - `GET /new_game_ai` - создание новой игры, компьютер делает первый ход
    - `POST /game/{x}` - ход игрока для игры с uuid = x, в теле запроса в формате json передаются координаты сделанного хода, Row и Col. Компьютер делает ответный ход и передает в клиент структуру с игрой в формате json
